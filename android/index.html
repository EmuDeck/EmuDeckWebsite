Step by step guide

Follow this guide if you are having problems using the installer

1 - Preparations

Go to your Android Preferences and look for "Install Unknown Apps"

Make sure your Browser has permissions enabled

Download and install Termux on your Android Device from this link. If you already have termux make sure it is version 117. we've found issues on other versions.

As soon as you have downloaded Termux give it app installation permissions too

2 - Installation

Open Termux and paste this code:

/bin/bash -c "$(curl -fsSL https://bit.ly/3ntlzL8)"

Just follow the instructions, make sure you read them carefully

If you want to store your roms in your SD Card, make sure it is inserted in your device

Select what device do you have, if the device is not listed just chose ANDROID.

If your Android phone has a SnapDragon 850 or better chose Odin

This will define which emulators are downloaded, we curate each system to use the best emulator options for each system

The installer will check if you have the recommended emulators for your system, we always try to use Retroarch Cores first instead of standalone emulators. But some systems like GameCube or Nintendo DS need standalone versions to work properly. Keep in mind that some of those standalone emulator like Drastic are paid emulators

Every time you install an emulator you have to open it first so the emu can create all the configuration folders. After that come back to Termux so the installer can continue

Some of these emulators like Dolphin MMJR will be automatically configured for your system, some others won't so you might need to manually configure them.

After all the emulators have been configured the installer scripts will then start downloading.

Everything the Installer does to your system is backup up before, so if you uninstall the Installer everything will be back to the way it was before.

Chose where you want to store your roms. This cannot be changed unless you reinstall EmuDeck for Android so chose wisely

Chose wheter you want 4:3 or 8:7 aspect ratio for SNES. This can be changed later anytime.

The installer will then start downloading all the RetroArch cores for your system

By default 4:3 systems are configured to use System Bezels so the game won't have side black bars. This option can be changed later anytime.

After that the Themes will start downloading

The setup is now almost complete! Now you need to load your roms

Eject now your SD Card from your Android device, insert it into your PC and copy your roms to the folder specified in the instructions

You need to copy/move your roms to that specific folder. You can't use your existing roms folders. You can then later point your Frontend of choice

Remember to install the downloaded RetroArch cores and to configure your emulators with your BIOS as the installer suggests

Don't delete Termux as this will delete all your roms too

By Now the Installer will have finished configuring your Android Device

Getting artwork for Pegasus Frontend

If you've choosed Pegasus as your frontend you need to scrap your roms so Pegasus shows artwork for all your games, since Pegasus does not have a built in rom scraper

You can select only a platform to scrap o just select ALL and let the system download the images for the games you have on your device

There are several scrap engines, if your roms come from a clean no-intro romset select RetroArch as it will be really fast. Don't use RetroArch if you don't have a no-intro rom set

We recommend to use Screenscraper too for those images RetroArch doesn't have( please create an account first on ScreenScraper )

The first artwork download process will be done by RetroArch, this scraper only downloads Box Arts

After a few minutes Retroarch will finish and ScreenScrapper will start. But before you'll need to insert your ScreenScraper.fr credentials. If you don't have an account you can create one from your device by selecting NO, that option will open screenscraper.fr on your browser

After you've provided the installer with your credentials the scrap will start.

The credentials are stored on device. We never have and never will have access to them.

After this last step you can Exit Termux, Launch Pegasus and start playing your favorite games!

3 - Updating the Installer & changing configurations

The installer will get more and more features over time, we provide a script that allows you to update the installer, this update process will add new systems, fix bugs, etc.

After the installation if you open Termux you'll see a menu that allows you to update the Installer, change some configurations you did on the installation proccess and even uninstall everything
